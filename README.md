# EnemyMarkers

This mod displays markers on the screen for each player
present in the raid.

## Disclaimer

This is a fork of [enemymarkers](https://gitlab.com/flir063-spt/enemymarkers) made for the sole purpose of scavenger runs
Changes:
Always active as a scav
Never marks PMCs
